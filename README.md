To run locally, do the usual:
I have done Two Way
1 Method:
  1. Create a Python 3.6 virtualenv

  2. Install dependencies::

      pip install -r requirement.txt

  3. Migrate Code:
      ./manage.py migrate

  4. Run the Server:
      ./manage.py runserver
  
  5. Open new Terminal
      python3 create_pharmacy.py
      python3 check_pharmacy.py 
      
  
2 Method:      
  1. Create a Python 3.6 virtualenv

  2. Install dependencies::

      pip install -r requirement.txt
      
  3.  python3 python_check_pharmacy.py
